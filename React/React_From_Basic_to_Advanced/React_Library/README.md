JSX / 컴포넌트의 상태 관리 / 렌더링 최적화 / 성능 확인 ..

필요 -> 검색 -> 사용법 파악 -> 적용

[필요]

- 특정 기능이 필요해짐(직접 구현하기에는 조금 벅참)
- 더 나은 도구가 필요해짐
- 트렌드에 대한 새로운 기술에 대한 호기심

[검색]

* 필요 그대로 검색
* github / npm trends / 프론트엔드 커뮤니티 / 기술 블로그
* 유사한 라이브러리들 후보로 비교 (기능 / 성능 / 사이즈 / 인기 / 최신 ...)

[사용법 파악]

* 공식 문서
* github 코드 검색 (실제 용례)
* 기술 블로그의 적용하기 예제

[적용]

* POC (Proof of concept)
* 프로젝트 일부에 적용
* 프로젝트 전체에 적용
