### HTML_왜_중요

<br />

브라우저를 켜고 주소를 입력 

=> 서버에게 해당 페이지를 보기 위해 필요한 리소스Resource를 보내주세요

<br />

맥은 cmd+shift+i

윈도우 F12

개발자도구의 `Network` 탭 보면, 페이지 새로고침 해보면

JS, CSS, ...

`Doc` <- HTML에 해당

<br />

웹 페이지의 모든 것의 시작은 `HTML`에서 비롯

이미지 파일, pdf 파일.. 을 브라우저로 열어보면 Elements에 html 태그 작성된게 보일 것임

웹 페이지에 보이는 모든 것들은 기승전 HTML 태그의 결과물

<br />

웹 브라우저 환경의 베이스가 되는 HTML

문서의 `구조`와 `정보 위계`가 명확하게 보이는 아름다운 HTML 코드를 작성하자

<br />

`Semantic Markup` 시맨틱하게 마크업을 한다

의미있게 작성하면 좋은점? 중요도 우선순위

검색 엔진 최적화 `Search Engine Optimization` 에도 도움됨
